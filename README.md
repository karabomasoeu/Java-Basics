# Java Basics

A collection of six beginner-friendly Java projects that cover fundamental programming concepts. Each project is designed to practice core Java skills through practical, hands-on applications.

## About This Repository

This repository contains console-based Java programs that demonstrate essential programming concepts like loops, conditionals, arrays, and basic object-oriented programming. These are perfect for learning and practicing Java fundamentals.

## Projects

### 1. Simple Calculator
Program performs basic arithmetic operations including addition, subtraction, multiplication, and division.

**What You'll Learn:**
- Working with user input
- Using conditional statements
- Handling division by zero errors

---

### 2. Number Guessing Game
An interactive game where the computer generates a random number and you try to guess it.

**What You'll Learn:**
- Using the Random class
- Implementing while loops
- Providing feedback based on user input

---

### 3. Student Grade Manager
A program to store student grades, calculate averages, and assign letter grades.

**What You'll Learn:**
- Working with arrays or ArrayLists
- Calculating averages
- Using if-else statements for grade assignment

---

### 4. To-Do List Application
Manage your tasks by adding, viewing, and removing items from a list.

**What You'll Learn:**
- Using ArrayList for dynamic data storage
- Creating menu-driven programs
- Performing add, remove, and display operations

---

### 5. Temperature Converter
Converts temperatures between Celsius and Fahrenheit.

**What You'll Learn:**
- Working with formulas and calculations
- Handling decimal numbers
- Creating bidirectional conversions

---

### 6. Bank Account System
Simulates a simple bank account with deposit, withdrawal, and balance checking features.

**What You'll Learn:**
- Basic object-oriented programming
- Managing state with variables
- Validating transactions (e.g., preventing overdrafts)

## Requirements

- A text editor or IDE (like IntelliJ IDEA, Eclipse, or VS Code)

## Skills Practiced

Through these projects, you'll practice:
- Variables and data types
- Conditional statements (if-else, switch)
- Loops (for, while)
- Arrays and ArrayLists
- Methods and functions
- Basic object-oriented programming
- User input handling with Scanner
- Error handling basics
