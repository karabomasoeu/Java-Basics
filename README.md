# Java-Basics

6 projects, as listed below, will be create:

Simple Calculator - Performs basic arithmetic operations (add, subtract, multiply, divide)
Number Guessing Game - Interactive game where you guess a random number between 1-100
Student Grade Manager - Manages student grades, calculates averages, and assigns letter grades
To-Do List - Add, remove, and display tasks using ArrayList
Temperature Converter - Converts between Celsius and Fahrenheit
Bank Account System - Simulates basic banking operations with deposits and withdrawals
